# FastAPIDemo
A Demo Notes app using Python's FastAPI and PostgreSQL


## Running the app
   `uvicorn main:app`

## Environment variables

   The following environment variables are required to run the app

   POSTGRES_USER=

   POSTGRES_PASSWORD=

   POSTGRES_HOST=

   POSTGRES_PORT=

   NOTES_DATABASE_NAME=

## Updating requirements.txt
    pip3 freeze -> requirements.txt
